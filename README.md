# Shabelle Bank USSD Simulator (Local Version)

This branch contains a **fully local** USSD testing environment with a browser-based phone emulator.

## Features
- No ngrok or Africa's Talking required
- Realistic phone UI with numeric keypad
- PIN masking (shows ●●●●)
- Auto-clear input after each step
- My Account menu (Balance + Mini Statement)

## How to Run

1. Run the Spring Boot application
2. Open http://localhost:8080 in your browser
3. Use the keypad or type and press **DIAL / SEND**

Test PIN: `1234`
